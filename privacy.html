FileSync プライバシーポリシー

最終更新日: 2026年5月3日 / 初版発行: 2026年5月3日

要約: FileSync は個人情報を一切収集しません。広告・追跡もありません。 各クラウドサービスへの認証情報やファイルのメタデータは、すべてあなたの端末内（Apple Keychain / Android KeyStore など）にのみ保存され、開発者のサーバーへ送信されることはありません。

1. はじめに

FileSync(以下「本アプリ」)は、ユーザーのプライバシーを最重要視しています。 本プライバシーポリシーは、本アプリがどのような情報をどのように取り扱うかを説明します。 本アプリは、複数のクラウドストレージ(Google Drive / OneDrive / Dropbox / Box / pCloud 等)、ネットワークプロトコル経由のサーバー(WebDAV / SFTP / SMB / FTP)、USB ストレージ、端末内ローカルストレージを統合管理するファイルマネージャーです。

2. 収集する情報

2.1 本アプリが「収集しない」情報

本アプリは、以下のいかなる情報もサーバー側で収集しません。

  • 個人を特定できる情報(氏名・住所・電話番号・メールアドレス等)
  • 操作したファイルの本体データ
  • 位置情報
  • 広告ID等のトラッキング識別子(IDFA / GAID)
  • 利用履歴・行動アナリティクス
  • クラッシュレポート・診断情報の自動送信

本アプリは外部のアナリティクス基盤(Google Analytics / Firebase Analytics / Amplitude 等)を一切使用していません。

2.2 端末内に保存される情報

以下の情報はあなたの端末内にのみ保存されます。開発者のサーバーには送信されません。

  • 各クラウドサービスの OAuth アクセストークン / リフレッシュトークン
  • WebDAV / SFTP / SMB / FTP の 接続先情報・認証情報(ユーザー名・パスワード・SSH鍵)
  • ファイル一覧(メタデータ)の一時キャッシュ
  • 同期設定・表示設定・お気に入り
  • バックアップ・転送ジョブの履歴

これらの情報は、iOS では Apple Keychain および暗号化されたアプリ専用ストレージに、Android では Android KeyStore および暗号化された内部ストレージに保存され、OSの保護機構によってアプリ単位で隔離されます。

2.3 ユーザー操作により外部送信される情報

ユーザーがバックアップ・アップロード・転送機能を使用した場合に限り、対象ファイルがユーザー自身が指定した送信先(クラウドストレージ・SMB / WebDAV / SFTP / FTP サーバー等)へ送信されます。 これらは ユーザーが明示的に選択したサービスとの直接通信であり、開発者のサーバーを経由しません。

3. 第三者サービス

本アプリは、ユーザーが接続を選択した以下のサービスと ユーザー名義で直接通信します。本アプリ開発者は、これらのサービスにユーザーの代理として接続するのみで、認証情報を開発者サーバーに送信することはありません。

  • Google Drive(Google LLC のプライバシーポリシー)
  • Microsoft OneDrive(Microsoft のプライバシーステートメント)
  • Dropbox(Dropbox のプライバシーポリシー)
  • Box(Box のプライバシーポリシー)
  • pCloud(pCloud のプライバシーポリシー)
  • WebDAV / SFTP / SMB / FTP サーバー(ユーザーが指定する任意のサーバー)

これらのサービスにおけるデータの取り扱いは、それぞれのサービス提供者のプライバシーポリシーに従います。

4. アクセス権限

本アプリが要求する OS 権限と、その用途は以下のとおりです。各権限は、対応する機能を使用する場合にのみ要求され、機能を使用しない限り収集・利用されません。

  • 写真ライブラリ: 写真・動画の自動バックアップ機能のため
  • ローカルネットワーク: SMB / WebDAV サーバー検出のため
  • Face ID / Touch ID / 生体認証: アプリロック機能のため
  • ファイル共有・ドキュメントブラウザ: ファイルのインポート・エクスポートのため
  • バックグラウンド処理: アプリを閉じている間の転送継続のため
  • 外部ストレージ(Android): USB / SDカード等のローカルファイルアクセスのため

5. Cookie・トラッキング

本アプリは Cookie を使用しません。また、以下の機能を一切使用していません。

  • 広告識別子(IDFA / GAID)の収集
  • クロスサイト・クロスアプリのトラッキング
  • 第三者広告ネットワークへの接続
  • App Tracking Transparency(ATT)対象の追跡行為

6. セキュリティ

  • 認証情報・トークンは iOS Keychain / Android KeyStore に保存され、OS のセキュアエンクレーブで保護されます
  • ネットワーク通信はすべて HTTPS / SMB3 / SSH / TLS 等の暗号化プロトコルを使用します
  • 生体認証(Face ID / Touch ID / 指紋)によるアプリロック機能を提供します
  • SFTP は公開鍵認証にも対応します

7. 子供のプライバシー

本アプリは13歳未満の子供から意図的に個人情報を収集することはありません。 そもそも本アプリは年齢を含むあらゆる個人情報を収集していないため、年齢に関わらず、開発者側で個人情報が記録されることはありません。

8. 国際的なデータ転送

本アプリ自体はデータを開発者サーバーに送信しません。 ただし、ユーザーが選択したクラウドサービス(Google Drive 等)は海外にデータを保管する場合があります。詳細は各サービスのプライバシーポリシーを参照してください。

9. GDPR / CCPA への対応

本アプリは個人データを開発者サーバーで収集・保管しないため、原則として GDPR におけるデータコントローラー、CCPA における事業者には該当しません。 ユーザーはアプリのアンインストール、または各設定画面からの接続解除により、端末内に保存された認証情報を削除できます。

10. データの削除

10.1 認証情報・接続設定

  • 各サービスの「ログアウト」または「接続解除」操作で、対応するトークンが端末から削除されます
  • アプリをアンインストールすると、Keychain / KeyStore を除くすべてのアプリデータが削除されます
  • Keychain / KeyStore 内の項目は OS の設定からも削除可能です

10.2 アップロード済みのファイル

各クラウド/サーバー側の管理画面または公式アプリから直接削除してください。本アプリ内からの削除操作にも対応しています。

11. プライバシーポリシーの変更

本プライバシーポリシーに重要な変更がある場合は、アプリ内通知または App Store / Google Play のアップデート情報、および本ページの「最終更新日」の更新によりお知らせします。

12. お問い合わせ

プライバシーに関するご質問やご不明な点がございましたら、本アプリの GitHub リポジトリの Issues よりお問い合わせください。

  • 連絡窓口: GitHub Issues
  • 運営: FileSync 開発チーム(個人開発者)

13. まとめ

FileSync は「ローカルファースト・プライバシー・バイ・デザイン」の原則に基づいて開発されています。

  • 開発者サーバーに認証情報を送信しません
  • ファイル本体は開発者サーバーを経由しません
  • 認証情報は端末内 Keychain / KeyStore で暗号化保護
  • 第三者へのデータ販売・提供は一切ありません
  • 広告・追跡なし
